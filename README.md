
# Widgets Config

Widgets configuration templates for GOTCHU

## Usage

Run the provided CLI tool to install the configurations for the specified engine.

```sh
  gotchu-widgets <engine>
```

## Installation


```sh
  npm i -g @gotchu/widgets-config
```


## Supported Engines

### Wigify

```sh
  gotchu-widgets wigify
```


### Übersicht

```sh
  gotchu-widgets uebersicht
```
